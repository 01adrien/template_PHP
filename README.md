# Mini Framework PHP

**Description**  
Ce projet est un **mini framework PHP maison**, développé à des fins pédagogiques.  
Il implémente les briques essentielles d’un framework web moderne : **routing, injection de dépendances, middleware**, et aussi **système léger d’entités / pseudo-ORM** pour gérer la persistance de données.  

L’objectif est de comprendre le fonctionnement interne d’un framework.

---

## Fonctionnalités

- **Routing** : gestion des routes HTTP simples et dynamiques  
- **Injection de dépendances** : conteneur léger pour instancier les services et contrôleurs  
- **Middleware** : traitement des requêtes avant d’atteindre les contrôleurs  
- **Système d’entités / pseudo-ORM** :
  - CRUD basique pour les entités
  - Mapping objet → données (fichier ou base légère)
- Architecture modulaire et claire  
- Gestion basique des requêtes et réponses HTTP
